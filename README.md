 🎓 CampusAI Pro
 
 Intelligent Student Assistant for Academic Success

CampusAI Pro is a desktop application developed using **Java Swing** that acts as an AI-powered student assistant. It helps students organize their studies, calculate GPA, manage notes, prepare for interviews, explore career opportunities, and build skills for studying or working in Japan.

This project was built as a portfolio project to demonstrate Java development, desktop application design, and problem-solving skills.



## ✨ Features

### 📊 Dashboard
- Displays academic overview
- Quick access to all modules
- Modern and responsive interface

### 📅 Study Planner
- Create daily study plans
- Add and manage tasks
- Save tasks locally

### 🎯 GPA Calculator
- Calculate semester GPA
- Simple and easy-to-use interface

### 📝 Notes Manager
- Create and save notes
- Organize study materials
- Local file storage

### 💼 Career Advisor
- Career guidance based on interests
- Technology learning roadmap
- Skill recommendations

### 🎤 Interview Practice
- Technical interview questions
- HR interview preparation
- Self-practice module

### 🇯🇵 Japan Mode
Special module designed for students planning careers in Japan.

Includes:
- JLPT Roadmap
- Japanese Learning Guide
- Career Information
- Skill Development Suggestions

### ⚙ Settings
- Application preferences
- User customization



## 🛠 Tech Stack

- **Language:** Java
- **GUI Framework:** Java Swing
- **IDE:** VS Code
- **Version Control:** Git & GitHub
- **Storage:** Local Files

## 🚀 Getting Started

### Prerequisites

- Java JDK 17 or later
- Visual Studio Code or IntelliJ IDEA

### Clone Repository

```bash
git clone https://github.com/priyadharshinis-28/CampusAI-Pro.git
```

### Open Project

```bash
cd CampusAI-Pro
```

### Compile

```bash
javac src/*.java
```

### Run

```bash
java -cp src Main
```
once you have completed running. kindly enter you username and password and register .

After registration, use the same username and password to login
